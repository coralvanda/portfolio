Neighborhood Map

This project required programming the UI and underlying structure for a single-page web app that utilizes the Google Maps API and GeoNames data to display interactive points of interest and related information on a Google Map. This project involved the use of Javascript, jQuery, KnockoutJS, HTML 5, CSS, Google Maps Javascript API, and AJAX requests to retrieve data from the GeoNames database.
